# Icon Information

The extension requires a 128x128 PNG icon named `icon.png` in the root directory.

## Icon Requirements

- **Size**: 128x128 pixels
- **Format**: PNG
- **Theme**: Should represent Spotify/Music
- **Colors**: Spotify green (#1DB954) recommended

## Creating the Icon

You can:
1. Design a custom icon with music/Spotify theme
2. Use a free icon from:
   - [Flaticon](https://www.flaticon.com/)
   - [Icons8](https://icons8.com/)
   - [Font Awesome](https://fontawesome.com/)

## Temporary Solution

For now, you can use any 128x128 music-related icon and name it `icon.png`.

## Example Icon Ideas

- Music note with VS Code colors
- Spotify logo style
- Headphones icon
- Play button with code brackets
- Waveform visualization

**Note**: Make sure you have the rights to use any icon you choose!
